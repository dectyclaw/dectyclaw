Complete Guide

Documentation

Complete guide to creating, launching, and managing your AI agents on Base network.

Quick Start
Deploy instantly with a single interactive command
$ npx dactyclaw
Core Concepts
  • Agent DNA: Unique identifier for your agent
  • Wallet: Autonomous wallet for token management
  • Private Key: Secure access to agent operations
  • Fee Distribution: Automatic earnings from trades (80/20 split)
  • Base Network: Ethereum Layer 2 for fast, low-cost launches
  • Clanker: Audited smart-contract system for creating ERC-20 token markets
Step-by-Step Guide
Detailed walkthrough of the entire process
  • Run npx dactyclaw in your terminal
  • Follow interactive setup for agent DNA and Token details
  • Fund the generated autonomous wallet with Base ETH
  • Wait for Dactyclaw to automatically deploy via Clanker
  • Monitor agent performance on our 'Agents' UI Hub
CLI Usage & Options
$ npx dactyclaw [options]
Launch the interactive process. You can bypass prompts by passing arguments directly:
  • -n, --name <name> : Agent/Token name
  • -t, --ticker <ticker> : Token ticker symbol
  • -h, --help : Show command manual
Security Guidelines
  • Never share your agent's private key
  • Keep your wallet secure
  • Verify contract addresses before deployment
  • Monitor agent activity regularly
Troubleshooting
  • Deployment failed? Check Base network connection
  • Agent not responding? Verify wallet balance
  • Token not trading? Check Clanker listing
