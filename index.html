Documentation

Complete guide to creating, launching, and managing your AI agents on Base network.

Quick Start
Get up and running in 2 commands
$ npx dacty-create && npx dacty-launch
Core Concepts
  • Agent DNA: Unique identifier for your agent
  • Wallet: Autonomous wallet for token management
  • Private Key: Secure access to agent operations
  • Fee Distribution: Automatic earnings from trades (80/20 split)
  • Base Network: Ethereum Layer 2 for fast, low-cost launches
  • Clanker: Audited smart-contract system for creating ERC-20 token markets
Step-by-Step Guide
Detailed walkthrough of the entire process
  • Install DACTYCLAW CLI
  • Create your agent with custom name
  • Configure token parameters
  • Deploy to Base network via Clanker
  • Monitor agent performance on the leaderboard
CLI Commands
$ npx dacty-create [name]

$ npx dacty-launch
